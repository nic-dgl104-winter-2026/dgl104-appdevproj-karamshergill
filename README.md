# DGL 104 - Task Management System (TMS)

## Introduction
This project is a simple Task Management System made for DGL 104.  
It is built using HTML, CSS, JavaScript, and PHP.  
The main purpose of this project is to help users manage their tasks in an easy way.  
A user can log in, add tasks, edit tasks, delete tasks, assign tasks, and check task details.

## Tech Stack
- PHP
- HTML
- CSS
- JavaScript
- Etc.

## Unique Features
- simple login system
- create new tasks
- edit existing tasks
- delete tasks
- assign task to a user
- set task priority
- add due date
- filter tasks by status
- filter tasks by priority
- show task summary counts
- notification message when task is updated
- localStorage used for saving task data in browser

## Design Patterns
- Factory Pattern
  It is used for creating user roles like Admin, Manager, Developer, and Tester.

- Observer Pattern 
  It is used in a simple way for task update notifications.

- Singleton Pattern
  It is used as a shared app state object so the task data can be managed from one place.

## Installation Guidelines
1. Download or clone this project repository.  
2. Open the project folder in VS Code or any code editor.  
3. Open the project in browser.
4. Start using the task management system. 


## Summary of the Project
This project is a small task management website for managing daily or team tasks.  
Users can add task title, description, priority, due date, and assigned user.  
They can also update task status and remove tasks if needed.  
The project is simple but useful for understanding CRUD(create,read,update and delete) operations, design patterns, and frontend/backend working together.
